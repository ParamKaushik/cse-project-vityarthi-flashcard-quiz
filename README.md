# Flashcard Quiz App

## Overview
The Flashcard Quiz App is a simple Python-based application designed to help students learn and memorize information through interactive quizzes. Users can create custom flashcards with questions and answers, take quizzes, and receive instant feedback on their performance. This project was developed as part of PYTHON ESSENTIALS to apply fundamental programming concepts including data structures, loops, conditionals, and user input handling.

## Features
- Store multiple flashcards with question-answer pairs
- Interactive quiz interface through command line
- Instant feedback on correct/incorrect answers
- Score tracking throughout the quiz
- Final score display at the end
- Simple and intuitive user experience
- Easily customizable flashcard content

## Technologies/Tools Used
- *Programming Language*: Python 3.x
- *Development Environment*: Visual Studio Code
- *Version Control*: Git and GitHub
- *Data Structures*: Dictionaries and Lists
- *Core Concepts*: Loops, Conditionals, Functions, User Input/Output
## Steps to Install and Run the Project

### Prerequisites
- Python 3.x must be installed on your system
- Download Python from [python.org](https://python.org) if not already installed

### Installation Steps
1. Clone or download this repository to your local machine
git clone https://github.com/ParamKaushik/flashcard-quiz-app.git
Or download as ZIP and extract

2. Navigate to the project directory
cd flashcard-quiz-app
3. Run the program
python pythonproject.py
### Test Case 1: Correct Answers
1. Run the program: python main.py
2. Answer all questions correctly
3. Expected Result: Score should show 100% or full marks

### Test Case 2: Incorrect Answers
1. Run the program
2. Provide wrong answers intentionally
3. Expected Result: Program should indicate incorrect answers and show reduced score

### Test Case 3: Mixed Answers
1. Run the program
2. Answer some correctly and some incorrectly
3. Expected Result: Score should accurately reflect the number of correct answers
### Test Case 4: Case Sensitivity 
1. Test if answers are case-sensitive
2. Try variations like "Python" vs "python"
3. Verify program handles it appropriately
