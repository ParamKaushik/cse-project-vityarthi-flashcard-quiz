# Flashcard Quiz Program — Project Statement

## Problem Statement
Many students struggle to memorize information effectively when studying. Traditional methods like writing notes or reading textbooks can become repetitive and boring.  
This project aims to solve this problem by creating an interactive **Flashcard Quiz Program** in Python, allowing users to test their knowledge in a fun and engaging way. The program provides immediate feedback, randomizes questions, and keeps track of user performance.

---

## Scope of the Project
This project focuses on building a **beginner-friendly, console-based quiz application** using only fundamental Python concepts such as:

- Dictionaries  
- Loops  
- Conditional statements  
- Random module  
- Time module  
- User input and output  

The scope includes:
- Adding custom flashcards  
- Asking questions randomly  
- Checking answers  
- Enforcing a time limit  
- Displaying total score and percentage  
- Clean and organized project files  

The project does **not** include advanced features such as databases, GUIs, or web interfaces.

---

## Target Users
The program is designed for:

### Students  
Who want to test their knowledge quickly and interactively.

### Beginners learning Python  
Because the code demonstrates basic concepts clearly.

### Teachers or tutors  
Who may want a simple tool to quiz students on custom topics.

### Anyone studying any subject  
Since flashcards can be customized for any type of content.

---

## High-Level Features
Below is a summary of the key features included in the program:

###  1. Add Custom Flashcards
Users can create their own question–answer pairs to expand the quiz.

###  2. Random Question Selection
The program uses the `random` module to shuffle questions, keeping the quiz unpredictable.

###  3. Timer for Each Question
Users must answer within a set time limit (e.g., 5 seconds), adding challenge and engagement.

###  4. Case-Insensitive Answer Checking
Uses `.lower()` to ignore capitalization mistakes, making the quiz more user-friendly.

###  5. Score Calculation
Tracks how many questions the user answered correctly and provides a final score percentage.

###  6. Clean Console Formatting
Uses spacing and newlines for easy readability.
